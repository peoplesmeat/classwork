SQLite Query Analyzer Documentation

SQLiteQA is a query analyzer tool for ADO.NET Data Provider for SQLite  . The current version supports sqlite version 2 & 3.  I currently do not plan on supporting or updating this app but I wanted to release the code so that it may help others. Please direct any SQLite.NET questions to the SQLite.NET forums. See below for a poorly written list of features.

SQLiteQA is released under the GNU GPL License.

Version 1.4.1 (Updated 2005.04.25)   -   See changes.txt for details
  • Released under GNU GPL.
  • New SQL Editor based on the ICSharpCode Text Editor.
  • Online Help added, defualts to http://www.sqlite.org/lang.html. This can be changed in the option file.
    Sample:
    <section name = "Help">
       <entry name="Online"> http://www.sqlite.org/lang.html</entry>
    </section>
  • Refresh option to reload the database and table changes. This is done manually.
  • Opens and edits SQLite 2 & 3.
  • Creates new databases of Version 3 only.
  • Improved query saving. Added "Save As" button and the "Save" button is a little less annoying.
  • Right click tables in the "Database Objects" list for some quick table options (select, insert, etc...).
  • Supports multiple result set queries.
